# i18n Support

The Markdown WYSIWYG Editor uses a host-driven i18n strategy.

- Provide a translator function `t(key)` via `I18nProvider`.
- UI labels are defined only by `I18N_KEYS`.
- If no translator is provided, keys are rendered as-is.

## Integration with i18n Libraries

### With react-i18next

```tsx
import { I18nProvider, MarkdownEditor, type Translator } from 'markdown-wysiwyg-editor';
import { useTranslation } from 'react-i18next';

function App() {
  const { t: t18n } = useTranslation();
  const [content, setContent] = useState('');

  const t: Translator = (key, fallback) => t18n(key, { defaultValue: fallback ?? key });

  return (
    <I18nProvider t={t}>
      <MarkdownEditor value={content} onChange={setContent} />
    </I18nProvider>
  );
}
```

### With next-intl

```tsx
import { I18nProvider, MarkdownEditor, type Translator } from 'markdown-wysiwyg-editor';
import { useTranslations } from 'next-intl';

function App() {
  const tIntl = useTranslations();
  const [content, setContent] = useState('');

  const t: Translator = (key, fallback) => {
    const value = tIntl(key);
    return value === key ? fallback ?? key : value;
  };

  return (
    <I18nProvider t={t}>
      <MarkdownEditor value={content} onChange={setContent} />
    </I18nProvider>
  );
}
```

## Translation Keys

The `I18N_KEYS` constant provides all translation keys:

```typescript
import { I18N_KEYS } from 'markdown-wysiwyg-editor';

// Table context menu keys
I18N_KEYS.table.rowOperations; // 'markdown_editor.table.row_operations'
I18N_KEYS.table.addRowAbove; // 'markdown_editor.table.add_row_above'
I18N_KEYS.table.addRowBelow; // 'markdown_editor.table.add_row_below'
I18N_KEYS.table.deleteRow; // 'markdown_editor.table.delete_row'
I18N_KEYS.table.columnOperations; // 'markdown_editor.table.column_operations'
I18N_KEYS.table.addColumnLeft; // 'markdown_editor.table.add_column_left'
I18N_KEYS.table.addColumnRight; // 'markdown_editor.table.add_column_right'
I18N_KEYS.table.deleteColumn; // 'markdown_editor.table.delete_column'
I18N_KEYS.table.deleteTable; // 'markdown_editor.table.delete_table'
I18N_KEYS.table.cancel; // 'markdown_editor.table.cancel'
```

    "table": {

      "add_row_below": "Add Row Below",
